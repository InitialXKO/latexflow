# LatexFlow

[English](README.md) | [中文](README_CN.md)

LatexFlow 是一款专为科研工作者和极客设计的安卓应用，旨在无缝衔接手写数学公式与 LaTeX 代码。通过手写识别和蓝牙 HID 模拟技术，它可以让你直接将公式“键入”电脑，无需安装任何客户端软件。

## 功能特性

- **无限画布：** 低延迟手写画布，支持压力感应和笔触优化。
- **实时识别：** 集成高性能 OCR（如 MyScript），将手写笔迹转换为 LaTeX 代码。
- **实时预览：** 使用 KaTeX 引擎即时渲染，确保公式准确无误。
- **蓝牙 HID “魔法”：** 手机模拟蓝牙键盘，无需电脑安装额外软件，配对即用。
- **一键注入：** 将公式代码直接发送到 Windows、macOS、Linux 或 iPadOS 的当前光标位置。
- **GeoGebra 数学键盘（新增）：** 集成 GeoGebra 官方 `keyboard-base` 逻辑，提供专业的符号和模板输入。
- **历史与收藏：** 自动保存识别记录，支持收藏常用复杂公式。

## 快速入门

1. **配对：** 打开应用，在电脑蓝牙设置中搜索并连接 “LatexFlow Keyboard”。
2. **书写：** 在手机画布上手写公式，或使用新增的数学键盘。
3. **核对：** 在屏幕底部的实时预览区检查识别结果。
4. **注入：** 点击电脑编辑器中的光标位置，然后在手机上点击“注入 (Inject)”。

## CI/CD 架构

本项目包含完善的 CI/CD 配置：

- **构建 (Build)：** 推送代码时自动生成 Debug APK。
- **发布 (Release)：** 推送标签 (`v*.*.*`) 时自动构建带签名的 Release APK 并创建 GitHub Release。
- **密钥生成：** 提供辅助工作流用于生成和管理签名密钥。

更多关于 CI/CD 的细节，请参阅我们的 [工作流文档](.github/workflows/README.md)。
